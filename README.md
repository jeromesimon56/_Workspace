# Workspace Agents

Ce workspace contient plusieurs agents IA configurés pour automatiser la gestion des emails Outlook et Gmail.

## Agents disponibles
- outlook_agent : récupération, tri, extraction pièces jointes Outlook
- gmail_agent : récupération, tri, extraction pièces jointes Gmail

## Structure
Voir folder_structure.json dans shared/templates.

## Dépendances
- PowerShell 7+
- Python 3.10+
- Modules : Microsoft.Graph, google-api-python-client
- VS Code extensions : Copilot Chat, Copilot Agents, PowerShell, Python
